Skills

HTML/CSS


HTMLとCSSだけで言えば、ほとんどのWebサイトのコーディングをすることができます。また、WordPressテーマを使用したカスタマイズもサイトのデザイン面であれば、ほぼ可能です。

Bootstrap


Bootstrapの基本的な使い方はある程度、理解しています。詳しくはポートフォリオをご覧ください。

Javascript/jQuery


JavascriptやjQueryはWebサイトを作るうえでの基本は理解しています。ハンバーガーメニュー、スライドショー、トップスクロール、アニメーションなどであれば実装可能です。 詳しくはポートフォリオをご覧ください。

WordPress


レンタルサーバーやドメインの準備、WordPressのインストールなどの使う前の準備やWordPressテーマの簡単なカスタマイズ、ローカルで作ったものをサーバーへアップロードするなどのことができます。 詳しくはポートフォリオをご覧ください（WordPressでのポートフォリオは1つのみです）。

PHP/Larabel


PHP/Larabelに関しては、オンラインスクールの学習過程で学びました。 PHPの基礎文法は理解しています。スクールでの制作物ですが、簡単なメッセージボードや簡単なTwitterのクローンを作りました。開発環境はAWS Cloud9です。 ログイン機能、投稿・削除機能、フォロー機能などがあります。

MySQL


MySQLに関してもオンラインスクールの学習過程で学びました。MySQLの接続・起動方法やデータベースのCRUD(データやテーブルの作成,削除など)基本的な操作は理解しているつもりです。メッセージボードやTwitterのクローンを作った時のデータベースにMySQLを使用しました。

Git/GitHub


Git/GitHubに関してもオンラインスクールの学習過程で学びました。メッセージボードやTwitterのクローンを作った時Gitでバージョン管理をしながら作り、それらをGithubにpushしました。 Gitでバージョン管理をする上での基本操作や、Githubの基本的な使いかたは理解しています。

Web業界に興味を持ったきっかけ

僕がWeb業界に興味をもったきっかけは、WordPressでブログをはじめるためにWebサイトを作ったことです。→（今から1年半前）

ブログ自体はすぐにやめてしまいましたが、そこでWebサイトの作り方やデザインに興味を持ち、HTML/CSSというものを知って、最初は本やオンラインサービスなどでそれらの基礎(Webサイトのつくり方)などの勉強しました。

最初はなんとなくで勉強していて、その頃、将来は何になろうかな程度に考えていたのですが、勉強しているうちにWebデザイナー、Webエンジニア(プログラマー)という職業を知り、そこでWebエンジニアになるにはどうしたらいいのか? どういうものなのか? などを調べつつ、オンラインスクールや独学でフロントエンドやバックエンドの勉強をしました。 その中でフロントエンドにやりがいを感じました。

将来目指すもの

就職の軸として現時点ではweb系・Web制作系の会社で働きたいと思っています。今までフロントエンドをメインで勉強してきたので業務でもフロントエンドに携わりたいです。ただ将来的には、フロントエンドだけでなくバックエンド、SEO、 Webマーケティングなどその分野において幅広く対応できるプロフェッショナルを目指しています。
